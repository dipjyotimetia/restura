# Chrome Web Store listing assets

Generated by [`../scripts/gen-store-assets.mjs`](../scripts/gen-store-assets.mjs)
(`npm run gen:store-assets --workspace @restura/extension`). All are 24-bit PNG
with **no alpha channel** — the Web Store rejects PNGs that carry alpha.

| File                      | Dashboard slot     | Size (px)  |
| ------------------------- | ------------------ | ---------- |
| `screenshot-1280x800.png` | Screenshots (≥1)   | 1280 × 800 |
| `small-promo-440x280.png` | Small promo tile   | 440 × 280  |
| `marquee-1400x560.png`    | Marquee promo tile | 1400 × 560 |

The store icon (128×128) is the extension's own `icons/icon-128.png` — no
separate asset needed. Edit the generator (copy/colors/layout) and re-run to
update; the brand mark and protocol-badge colors are pulled from the same
cobalt palette the side panel uses.
